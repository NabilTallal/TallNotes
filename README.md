# TallNotes : Real-Time Collaboration Notes App (WIP)

A real-time collaborative notes application built using the MERN stack.
Multiple users can edit the same document simultaneously with live updates.

## Project Status
Work in Progress  
Core features and backend architecture are still being developed.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Socket.io (real-time syncing)
- Quill.js or TipTap (rich text editor)

## Planned Features

### MVP
- Create, edit, and delete documents
- Real-time collaboration
- Auto-save on changes
- User authentication (JWT)
- Basic document sharing (link-based)

### Future Enhancements
- Role-based access control
- Version history
- Comments
- Activity log
- Dark mode
- Export to PDF/Markdown
